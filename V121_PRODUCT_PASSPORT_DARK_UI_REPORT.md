# V121 Product Passport Dark UI

Built from `rummy500-v120-ui-studio-colors-typography.zip`.

## What changed
- Restyled the app around the uploaded Product Passport reference.
- Removed all image/background-picture usage from the UI.
- Switched the visual system to dark passport-style panels, cobalt linework, technical labels, thin grids, and document-like score cards.
- Kept the v120 UI Studio and rewired its default values to the new passport palette.
- Added a new UI Studio storage prefix so old saved v120 color values do not override the new default look.
- Updated app theme/manifest colors.

## Functionality kept
- Scoring logic
- Add round
- Quick score buttons
- Negative toggle
- Closed tick box
- Penalty buttons
- Round history popup
- Settings/game setup
- UI Studio controls
- Supabase sync
- Four-player scroll/fit behavior
