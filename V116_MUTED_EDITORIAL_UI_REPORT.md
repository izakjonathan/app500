# V116 Muted Editorial UI

Built from v115 floating bottom controls.

## Design direction
- Replaced photographic/glass-heavy look with a muted sage/lilac/mauve interface inspired by the uploaded reference.
- Added large compressed title treatment and small pill controls.
- Rebuilt the scoreboard visually as black rounded data blocks with progress fill panels.
- Added soft editorial background motifs: diagonal hatching and dot cluster.
- Reworked bottom scoring controls into flat floating panels with black primary actions.
- Restyled modals/UI Studio to match the new muted interface while keeping existing functionality.

## Preserved
- Existing scoring logic.
- Add round, penalties, closed toggle, starter toggle.
- Settings, game setup, rounds overview, UI Studio, history, cloud sync.
- v115 fullscreen and floating bottom-control behavior.
