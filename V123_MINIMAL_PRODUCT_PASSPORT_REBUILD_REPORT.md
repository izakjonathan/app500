# V123 Minimal Product Passport Rebuild

Built from v122.

Design changes:
- Simplified the layout to remove the big empty right-side passport area.
- Reduced top scoreboard height and made the player cards compact.
- Kept the Product Passport direction: warm paper surface, cobalt technical type, thin rule lines, black outer background.
- Removed visual complexity from the dock controls while keeping all controls available.
- Kept UI Studio, round history, settings, scoring, closed player tick, quick buttons, penalties, save/sync, and new-game flow.
- Changed UI Studio storage namespace so old v122 local UI overrides do not pollute the new layout.

Build: `npm run build`
